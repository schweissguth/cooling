Technical Specs

  • Stocking
    • target lead time 3-4 weeks once stocked in local DCs
    • NAM: scheduled for release Q1-Y2020
  • Climate
    • -35C to +48C (with extreme low temp kit) -15C to +48C (without extreme low temp kit)
  • Placement / mounting
    • 50Hz version: bottom of rack only, open, semi-sealed, or sealed racks
    • 60Hz version: bottom mount is standard, top of rack can be achieved with optional stacking kit
  • Piping distance
    • 20m equiv +/-5m elevation.  Requires penetration through the rack to remote condensing unit
  • Power input
    • 50Hz version: 220-230V, 1ph, 50Hz
    • 60Hz version: 208-240V, 1ph, 60Hz
  • Electrical connection
    • 50Hz version: hardwired
    • 60Hz version: hardwired or optional plug
  • Condensate mgmt
    • standard gravity drain, optional external pump
  • Installation
    • Into rack onsite or via integrator.  Remote condensing unit requires mechanical contractor to install piping
  • Dimensional / Weight
    • Indoor
      • Height: 5U
      • Unit width: Can be installed in 19 inch rack
      • Depth: total 752mm
        • 672mm for cooling unit
        • Plenum dimension 80mm
      • Weight: 26kg
    • Outdoor
      • Height: 617mm（±2mm)
      • Depth: 296mm（±2mm)
      • Width: 856mm（±2mm)
      • Weight: 48kg
  • Certification & compliance
    • CCC
    • ETL
  • Controls & Firmware
    • onboard controller - controls fan and compressor realtime - Active Response Controls
  • Restart time
    • Upon power failure and reset, unit shall automatically resume cooling within 180s
  • Heat rejection and refrigerant
    • R-410a precharged
      • Precharge: Outdoor unit 1.65kg, Indoor unit 0kg
    • Split DX (remote condensing unit)
  • Capacity modulation
    • Variable speed rotary compressor
  • Energy efficiency
    • 50Hz version: 2.6 EER
    • 60Hz version: 2.8 EER
    • includes power consumption for all fans, electronics, and compressor both indoor and outdoor
  • Airflow or CFM/kW
    • 750 m^3/h (450 CFM).  Vertical discharge plenum cascades cold air vertically.  130CFM/kW
    • Air direction: Return air: rear of unit, Supply air: upflow
    • Requires blanking U space panels to prevent air recirculation
  • Cooling capacity
    • ~800W to 3.5kW L35-35
  • Humid + reheat
    • none (heat compensation only for compressor low load)
  • Power redundancy
    • none
  • Cooling redundancy
    • none
  • Sound
    • Indoor 68dBA/Outdoor 55dBA (1m away from discharge side @ 1m height, max fan speed, max compressor speed)
  • Appearance
    • Black color, gray vertical condensing unit for low footprint
  • User interface
    • optional rackmounted user interface for local control
    • without UI unit will start up and run at default
  • Communications
    • Modbus RTU 485 serial for remote control
  • IP/NEMA rating
    • IPX4 outdoor
    • IPX0 indoor
  • Storage temps
    • -20C ~ 52C   RH: 15%~85% (Unit storage range (indoor & outdoor units))
  • Retrofit/3rd party
    • can be used in "open" rack environments if there is space in rack available and a mechancial installer to perform the installation
